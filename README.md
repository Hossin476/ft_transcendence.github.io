## 🚀 Blast Off in 3... 2... 1...

1. Clone this bad boy:
   ```bash
   git clone https://github.com/yourusername/ft_transcendence.git
   cd ft_transcendence
   ```

2. Fire up the engines:
   ```bash
   make
   ```

3. 🌐 Open your favorite browser and navigate to `http://localhost:8000`

4. 🎉 Enjoy the awesomeness!
